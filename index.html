Masters-Thesis Information

The thesis revolves around evaluating the collision avoidance capabilities of a path planning algorithm called MCTSVO.

The capabilites of the algoritm is evaluated in the context of autonomous mobile robots (AMRs) used in automated warehouses.

Unity will be used to compare the MCTSVO algorithm with standard collision avoidance algorithms for a number of scenarios.
